**EcoRoute Explorer** is a modern, responsive Single Page Application (SPA) designed to help tourists and locals make smart, environmentally-aware travel decisions. By leveraging real-time weather and air quality data, the dashboard generates dynamic, condition-based itinerary recommendations.

## ✨ Features

* **Real-Time Environmental Data:** Fetches live temperature, weather conditions, and Air Quality Index (AQI) data using the OpenWeatherMap API.
* **Smart Recommendations Algorithm:** Dynamically generates travel advice based on strict environmental parameters (e.g., suggesting indoor activities during high AQI or extreme heat).
* **Modern UI/UX:** Features a sleek "Glass-morphism" design over a CSS-generated tech grid, complete with smooth micro-animations and a loading splash screen.
* **SPA Architecture:** Implements seamless tabbed navigation without page reloads, mimicking the feel of modern frontend frameworks using Vanilla JavaScript.
* **State Management:** Utilizes `localStorage` for persistent user sessions and a simulated login experience.
